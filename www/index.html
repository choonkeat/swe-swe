github npm docs architecture

Quick Start

Requires Node.js.

alias swe-swe='npx -y swe-swe'

Installs to /usr/local/bin (writable) or ~/.local/bin (fallback).

curl -fsSL https://raw.githubusercontent.com/choonkeat/swe-swe/main/install.sh | sh

Then

  1. cd /path/to/project
  2. swe-swe init # see docs for more options
  3. swe-swe up
  4. Open http://localhost:1977

Requirements

  • Docker & Docker Compose
  • Terminal access (macOS, Linux, Windows with WSL)
Works with Claude, Codex, OpenCode, Gemini, Aider, Goose
